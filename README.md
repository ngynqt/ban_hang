# Cửa hàng mẫu (Static)

Mở file `bang_hang.html` trong trình duyệt (double-click hoặc `Open File`) để chạy trang bán hàng mẫu. Đây là một demo frontend tĩnh không có backend.

Tính năng:
- Hiển thị danh sách sản phẩm
- Thêm/xóa thay đổi số lượng trong giỏ
- Lưu giỏ hàng vào `localStorage`
- Thanh toán mô phỏng (thông báo)

Muốn tiếp theo:
- Thêm backend (Node/Express, PHP, v.v.) để lưu đơn hàng
- Thêm quản trị để quản lý sản phẩm
